# Godot_Breakout
 A Classic Breakout clone made with Godot 4.

## Implemented
- paddle
- ball
- bricks
- scoreboard
- screen boundaries

## KEYs
- **SPACE** (Start: Start round)
- **ESC** (Restart: Restart Game)
- **A** or **LEFT_ARROW** (Paddle Left Movement)
- **D** or **RIGHT_ARROW** (Paddle Right Movement)
